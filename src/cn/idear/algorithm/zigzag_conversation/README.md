ZigZag Conversion
-----------------

## 分析

这道题是找规律。

n = 3:

0   4
1 3 5 7
2   6

n=4:

0     6
1   5 7
2 4   8 10
3     9

n=5:

0       8
1     7 9
2   6  10
3 5    11 13
4      12

首先不考虑插入列，横着看，则发现每行都构成一个等差数列，其公差d=(n-1)*2。所以可以推出第i行的计算公式：

a_i_j = i + d * j = i + 2 * (n - 1) * j, (j >= 0, i >= 0 && i <= n - 1)
 
然后对于插入列，第i行插入列的首项为2n - 2 - i，公差为(n - 1) * 2。所以可以推出计算公式为：

a_i_j = 2n - 2 - i + 2 * (n - 1) * j = 2 * (n - 1) * (j + 1) - i, (j >= 0, i > 0 && i < n-1)。（第0行和第n-1行没有插入项）
