Longest Valid Parenthesis
-------------------------

这个题目有些难。

需要2个数据结构。

* `stack`： 一个栈用以模拟匹配过程。

* `count`数组： `count[i]`记录从距离栈顶第`stack.size() - i - 1`个位置上最长匹配括号长度。

eg:

```
s       ( ) ( ) (
count   4 0 2 0 0
```

算法流程：

1. 初始化`count`数组元素为0

2. 发现`(`，压栈。

3. 发现`)`，如果栈顶是:
    
    * `)`: 压栈。
    
    * `(`: 这时说明发现新的匹配对。先退栈，然后`count[stack.size()] += count[stack.size() + 1] + 2`（即更新从当前栈顶位置开始的最长
    串长度）， 并将`count[stack.size() + 1]`清零，与最大值进行比较。
    
    * `null`: 这时栈为空，将`)`压栈。
